# 智能路亚钓鱼建议系统

基于DeepSeek大模型的专业级垂钓方案定制系统，使用Next.js构建。

## 功能特点

- 基于环境参数生成专业钓鱼建议
- 详细的分析和建议，包括装备优化、拟饵策略等
- 响应式设计，适配各种设备
- 支持多种鱼种和环境条件

## 本地部署指南

### 前提条件

- Node.js 18.0.0 或更高版本
- npm 或 yarn 包管理器
- DeepSeek API 密钥
